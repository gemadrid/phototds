# PhotoTDS

Aplicación de red social básica para compartir imágenes, desarrollada como proyecto de la asignatura Tecnologías de Desarrollo de Software de 3º de Ingeniería Informática de la Universidad de Murcia, curso 2022-2023.

## Funcionalidades

- Registro e inicio de sesión de usuarios
- Subida de fotos
- Álbumes
- Comentarios y "me gusta"
- Notificaciones
- Búsqueda de usuarios y publicaciones

## Tecnologías utilizadas

- Java
- Swing
- JUnit
- Arquitectura en 3 capas (presentación, dominio, persistencia)
- Persistencia mediante DAO
